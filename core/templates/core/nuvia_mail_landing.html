Parametri IMAP/SMTP da avere pronti

  • Indirizzo email aziendale completo
  • Server IMAP e porta (es. 993 SSL)
  • Server SMTP e porta (es. 465 SSL / 587 STARTTLS)
  • Username e password/app password
  • Firma email aziendale ufficiale
