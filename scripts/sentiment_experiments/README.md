# Esperimenti sentiment (script ad-hoc)

Script standalone usati per provare i modelli di sentiment analysis.
**Non sono test automatici**: erano in root come `test_sentiment*.py` e venivano
raccolti per errore da `manage.py test` (con una chiamata di rete all'import che
faceva fallire la discovery). Spostati qui e rinominati per toglierli dalla suite.

Eseguibili manualmente: `python scripts/sentiment_experiments/sentiment_v1.py`
